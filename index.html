Research & Projects

EQUATOR: A Deterministic Framework for Evaluating LLM Reasoning with Open-Ended Questions (Project Link)

  • EQUATOR presents a deterministic evaluation framework designed to assess the reasoning capabilities of Large Language Models (LLMs) on open-ended questions. The framework leverages automated evaluation pipelines powered by smaller LLMs like LLaMA 3.2B, ensuring scalable and efficient testing.
  • To broaden accessibility, plans were initiated to develop a collaborative website for sharing evaluations, benchmarks, and insights, aiming to support researchers and practitioners in advancing the reliability and performance of LLMs. This project establishes a foundation for transparent and standardized evaluation practices in the AI community.

Enhanced Momentum with Momentum Transformers (Project Link)

  • Momentum Transformer extending its application from futures and equity indices to equities, leveraging a hybrid architecture that combines attention mechanisms with LSTMs to capture long-term dependencies and adapt to evolving market conditions. The model successfully demonstrated its capability to enhance performance in dynamic trading environments while providing interpretable and intelligent decision-making insights.

Audio Classification of Normal Traffic and Siren Sounds: Spectrogram Encoding and Raw Audio Feature Extraction Approaches (Project Link)

  • AI-powered adaptive traffic system to classify emergency vehicle sirens from general road noise using audio data transformed into acoustic features and spectrograms. The system achieved exceptional accuracy, with fine-tuned ResNet18 correctly identifying all emergency and road noise samples, demonstrating its potential to enhance urban traffic management and emergency response efficiency.

Fine Tuning and Evaluating Large Language Models for HTML Code Generation (Project Link)

  • Fine-tuned the Falcon 7b language model to generate HTML code from natural language prompts, optimizing memory usage and computational efficiency through advanced techniques like BitsAndBytes quantization. Leveraged LoRA (Low-Rank Adaptation) for fine-tuning, employing gradient checkpointing and mixed-precision training to enhance the model's ability to translate natural language into accurate HTML code. The model's effectiveness was evaluated through exact match comparisons, ensuring high-quality output.

Sentiment Analysis using VADER – Price returns vs Sentiment (Project Link)

  • Developed an end-to-end data pipeline to analyze stock market sentiment by web scraping over 100 daily news articles from Finviz using requests and BeautifulSoup. Integrated the VADER tool to visualize sentiment trends across multiple trading days and utilized the yFinance API to automate historical price returns extraction, uncovering a strong correlation between sentiment shifts and stock price movements.
